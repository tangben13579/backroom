后室 Wiki - 帮助中心
 
欢迎来到后室 Wiki 帮助中心！这里将为你提供从入门到创作的完整指南，帮助你更好地浏览、编辑和贡献内容。
 
 
 
🔍 如何浏览 Wiki
 
1. 层级导航
 
- 主入口：通过首页 <a href="./level.md">层级</a>可按编号快速查找目标层级。
- 跳转方式：每个层级页面底部会提供「上一层/下一层」「关联层级」链接，方便你在不同层级间穿梭。

2. 页面结构说明
 
标准层级页面包含以下模块，方便你快速获取信息：
 
- 基础信息：生存难度、安全状态、层级定位等核心数据。
- 环境描述：层级外观、空间规则、特殊现象。
- 实体与资源：存在的实体类型、可获取的物资、危险提示。
- 入口与出口：进入/离开该层级的已知方法。
- 发现记录：流浪者的探索日志、历史事件。
 
 
 
✏️ 如何编辑与贡献内容
 
1. 编辑权限
 
- 所有注册用户均可编辑已有页面的补充内容（如修正描述、补充发现记录）。
- 创建新层级/实体页面需通过「新人审核」，提交后由管理员在24小时内审核。
- 禁止直接修改他人原创内容，需在Q群里提出修改建议并获得作者同意。
 
2. 内容规范
 
- 真实性原则：所有内容需符合后室世界观，禁止引入无关设定（如其他作品角色、现实科技）。
- 格式规范：必须使用 Wiki 标准 Markdown 模板（见下方 <a href="./template.md">模板</a>），保证页面排版统一。
- 文明创作：禁止包含低俗或煽动性内容，尊重其他创作者的劳动成果。
 
3. 模板参考（层级页面）
 
```markdown
  
# Level XX - "层级名称"

## 基础信息
| 项目 | 内容 |
| :--- | :--- |
| **生存难度** | 等级 X |
| **安全状态** | 简要描述安全/危险程度 |
| **层级定位** | 后室第X层/未知区域 |

## 环境描述
（详细描述层级的外观、空间规则、特殊现象，如「无尽走廊」「时间流速异常」）

## 实体与资源
- **实体**：列出存在的实体类型、行为模式、威胁等级。
- **资源**：可获取的物资（如杏仁水、能量棒）、特殊物品。

## 入口与出口
- **入口**：从哪些层级/方式可进入该层级。
- **出口**：离开该层级的已知方法。

## 发现记录
（记录流浪者的探索日志、历史事件，需标注发现者与日期）
 
 ```
 
 
⚠️ 常见问题与解决
 
Q1：页面显示异常（如代码块未闭合、标题错乱）
 
- 检查是否存在未闭合的 Markdown 符号（如 ```、|、#）。
- 对比 <a href="./template.md">模板</a> 修正格式，或在Q群里求助管理员。
 
Q2：提交新页面后未通过审核
 
- 可能原因：格式不规范、内容与世界观冲突、存在敏感信息。
- 查看QQ里审核反馈，按要求修改后重新提交。
 
Q3：找不到想要的内容
 
- 在Q群里留言，管理员/创作者会优先处理。
 
Q4：发现错误/恶意内容
 
- 联系Q群里的管理员或群主
- 管理员将在12小时内处理并反馈结果。
