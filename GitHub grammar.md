**GitHub语法**
# 1. 标题
 
源码： \# 一级标题 （#的左边不能出现任何文字）

效果： 

# 一级标题
 
源码： \## 二级标题 （#的左边不能出现任何文字）

效果： 

## 二级标题
 
 
 
# 2. 粗体 / 斜体
 
源码： \*\*粗体\*\* 

效果：粗体
 
源码： \*斜体\* 

效果：斜体 
 
源码： \~\~删除线\~\~ 

效果：删除线
 
 
 
# 3. 列表
 
源码： \- 列表项 

效果：- 列表项
 
源码： 1\. 有序项 

效果：1. 有序项
 
 
 
# 4. 引用
 
源码： \> 引用 （>的左边不能出现任何文字）

效果：

> 引用
 
 
 
# 5. 行内代码
 
源码：  （需要把 ` 放在代码或文字的左和右边）

效果： `代码`
 
 
 
 # 6. 链接
 
源码：\<a href="放置链接">名称</a>

效果： <a href="DY.jpeg">抖音号</a>
 
 
 
# 7. 图片
 
源码： \<img src="放置文件图片或者链接">

效果：<img src="./images level 0/OGLevel0.jpg">
 
 
 
# 8. 表格
 
源码：
 
plaintext
  
\| 左 | 右 |
\| -- | -- |
 
 
效果：
   
| 左 | 右 |
| -- | -- |
