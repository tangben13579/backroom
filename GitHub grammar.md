**GitHub语法**
1. 文本格式 (Text Formatting)

粗体: \**加粗文本** 或 \__加粗文本__  →  **加粗文本** 或 __加粗文本__

斜体: \*斜体文本* 或 \_斜体文本_  →  *斜体文本* 或 _斜体文本_

删除线: \~~删除内容~~  →  ~~删除内容~~

行内代码: \`code\`  →  `code`

2. 列表 (Lists)

无序列表: 使用 -, *, 或 +

有序列表: 使用数字加点 1., 2.

任务列表 (GitHub 特有):

\- [x] 已完成任务  →  - [x] 已完成任务

\- [ ] 未完成任务  →  - [ ] 未完成任务

\``` [1, 3]\  →  ``` [1, 3]
